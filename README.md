### Clone & install

**Please use yarn**

```bash
$ git clone git@github.com:mohfunk/seel.git
# or
$ git clone https://github.com/mohfunk/seel.git

$ cd seel
$ yarn
```

### Build & Run

##### **Build for development**

```bash
$ yarn dev
```

##### **Start development build**

```bash
$ yarn start
```

##### **Build for production**

```bash
$ yarn pro
```
